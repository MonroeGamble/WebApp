Customization Options

You can customize the feed by editing the configuration in the HTML file:

Available Settings

  • scrollSpeed: Speed of scrolling animation in seconds (default: 60)
  • categories: Filter articles by category
  • maxArticles: Maximum number of articles to display (default: 50)

Example Configuration

const CONFIG = { dataUrl: '../data/franchise_news.json', scrollSpeed: 45, // Faster scrolling categories: ['Trade press and industry news'], // Only trade press maxArticles: 30 // Limit to 30 articles };

Available Categories

  • Trade press and industry news
  • Big portals, directories, and lead-gen sites
  • Research, reviews, and investor-oriented info
  • Associations and policy hubs
  • Consultants, suppliers, and franchisor service firms

Technical Details

Data Source

The feed pulls data from a JSON file that is automatically updated every 6 hours via GitHub Actions. The data includes:

  • Article title and summary
  • Publication date and source
  • Category classification
  • Direct link to full article

Browser Support

The widget works in all modern browsers:

  • Chrome/Edge 90+
  • Firefox 88+
  • Safari 14+
  • Mobile browsers (iOS Safari, Chrome Mobile)

Performance

  • Load Time: <1 second on 3G connection
  • File Size: ~15KB (HTML + CSS + JS)
  • Data Size: ~3-5KB (compressed JSON)
  • Auto-Refresh: Every 6 hours
